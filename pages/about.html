About VentSnap

Open-source solutions for everyday HVAC problems.

The Origin Story

VentSnap started the way many projects do—with a kid who wanted a blanket fort and a parent who wanted it to work better.

The problem with blanket forts is keeping them up. Chairs fall over, clips don't grip, and the whole thing collapses right when the hot chocolate arrives. But there's this unused resource right there in the floor: the HVAC vent. What if the vent could be an anchor point? What if the warm air could inflate the fort like a gentle bounce house?

The Sheet Fort Clip was born. Then came the inevitable question: what else could you attach to a floor vent? The Round Duct Adapter followed shortly after, solving a completely different problem—redirecting airflow to a cold home office.

Open Source Philosophy

Everything here is open source under the MIT license. Download it, modify it, sell prints of it, fork it, improve it. We believe in the power of shared solutions.

The designs are provided as both STL (ready to print) and STEP (ready to modify) files. If you create an improvement or new attachment, we'd love to see it. Submit a pull request on GitHub or just share it with the community.

Design Principles

Print-Friendly First

Every design is optimized for FDM printing without supports. We don't assume everyone has a multi-material printer or wants to spend hours on post-processing.

Standard Components

When hardware is needed, we use standard sizes available at any hardware store. No exotic fasteners, no special order parts.

Real-World Tested

Every design gets tested in actual homes before release. We print prototypes, install them, use them for weeks, and iterate until they work reliably.

Material Appropriate

We specify materials based on actual operating conditions, not just what prints easiest. If PETG is required, we say so and explain why.

Contributing

There are several ways to contribute to VentSnap:

  • Submit bug reports if a design doesn't fit or function as expected
  • Share improvements via pull request on GitHub
  • Create new attachments using the standardized base interface
  • Write documentation or improve existing guides
  • Share your builds to help others see what's possible

Roadmap

Current priorities for future development:

  • 6" Round Adapter: Larger duct connection for high-flow applications
  • Y-Splitter: Split one vent into two smaller ducts
  • Adjustable Damper: Control airflow with an integrated blade
  • Wall Vent Versions: Adapt the designs for sidewall registers
  • Metric Sizing: Versions sized for European/Asian standard vents
