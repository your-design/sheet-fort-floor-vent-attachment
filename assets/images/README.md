# Asset Placeholders

This directory structure expects the following files. Add your actual product images here.

## Required Images

### Sheet Fort Clip
- `sheet-fort-clip-hero.jpg` - Main product photo (recommended: 800x800px)
- `sheet-fort-clip-installed.jpg` - Installed on a vent
- `sheet-fort-clip-detail.jpg` - Close-up of clip mechanism
- `sheet-fort-in-use.jpg` - Kids using a sheet fort

### Round Duct Adapter
- `duct-adapter-hero.jpg` - Main product photo (recommended: 800x800px)
- `duct-adapter-installed.jpg` - Installed on a vent
- `duct-adapter-with-hose.jpg` - With flex duct attached
- `duct-adapter-underside.jpg` - View of base/friction fit

### General
- `hero-placeholder.jpg` - Site hero image (optional)

## Recommended Image Specs

- Format: JPEG or WebP
- Hero images: 1200x600px minimum
- Product photos: 800x800px (square)
- Thumbnails will be auto-generated from main images
- Keep file sizes under 500KB for web performance

## Tips

1. Photograph on a clean, neutral background
2. Include scale reference (hand, ruler, or common object)
3. Show installation steps if possible
4. Capture "in use" photos for lifestyle appeal
