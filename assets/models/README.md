# 3D Model Files

Place your STL, STEP, and 3MF files here.

## Expected Files

### Sheet Fort Clip
- `sheet-fort-clip.stl` - Print-ready mesh
- `sheet-fort-clip.step` - Editable CAD file
- `sheet-fort-clip.3mf` - Bambu Studio project (optional)

### Round Duct Adapter  
- `duct-adapter-4in.stl` - Print-ready mesh
- `duct-adapter-4in.step` - Editable CAD file
- `duct-adapter-4in.3mf` - Bambu Studio project (optional)

## File Naming Convention

Use lowercase with hyphens. Match the filenames referenced in `_products/*.md` files.

## Adding New Products

1. Add your model files here
2. Create a new product page in `_products/`
3. Reference the files in the `downloads` front matter
